◆ initialize()

void Sgt::SimComponent::initialize ( )
finalvirtual

Initialize state of the object.

This simply does the initial work needed to set the object up, prior to simulation. The time following initialization will be set to negative infinity, and the object will be considered to be in an invalid state. To progress to a valid state, the object will need to undergo an update().

◆ validUntil()

virtual Time Sgt::SimComponent::validUntil ( ) const
inlinevirtual

When is the component scheduled to update?

An update may occur before this time if the component undergoes a contingent update. If this occurs, it is possible that validUntil() could change to a later time.

Reimplemented in Sgt::TapChanger, Sgt::SimpleBuilding, Sgt::Battery, and Sgt::Heartbeat.
