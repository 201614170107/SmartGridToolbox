Constant admittance component.

Assumed to be an upper triangular matrix. The diagonal elements are from the bus terminals to ground, And the off-diagonal elements are between pairs of bus terminals.

Constant current component.

Assumed to be an upper triangular matrix. The diagonal elements are from the bus terminals to ground, And the off-diagonal elements are between pairs of bus terminals. The phase is assumed to be relative to the phase of V, so that the power is independent of the absolute phase.
